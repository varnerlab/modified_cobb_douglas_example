ConstrainedCobbDouglas.compute_lambda — Function
compute_lambda(short_ema, long_ema; θ = 0.5) -> Vector{Float64}

Regime-lens λ from EMA crossover. λ_t = 1 / (1 + exp(-(short-long)/θ)) (sigmoid). The returned vector is a time series of λ values aligned with the input EMA series (one λ per timestep); callers index it at the decision time of interest.

source

compute_preference_weights(sim_parameters, tickers, gm_t, lambda) -> Vector{Float64}

No-news variant of the lectures function (spec §3.3). γi = tanh(αi/|β|^λ + |β|^(1-λ) · gm_t)

ConstrainedCobbDouglas.ewls_init — Function
ewls_init(α₀, β₀, σ_ε₀; half_life = 63.0, prior_weight = 63.0) -> MyEWLSState

Initialize EWLS state from a prior (α₀, β₀, σε₀). Decay η = 2^(-1/halflife); prior_weight seeds the sufficient statistics so EWLS recovers (α₀, β₀) exactly on day 0.
